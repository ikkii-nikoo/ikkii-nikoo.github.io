Creating Life - A Snake

Level 4: Professional Level

Snake Game Code Explanation

This is an implementation of the classic Snake Game using HTML, CSS, and JavaScript. Below is a detailed explanation of how the code works.

HTML Structure

HTML Structure
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Snake Game</title>
        <style>
            /* CSS code here */
        </style>
    </head>
    <body>
        <div id="game-container">
            <div id="game-board"></div>
            <div class="score">Score: <span id="score">0</span></div>
        </div>
        <script>
            /* JavaScript code here */
        </script>
    </body>
</html>

  1. DOCTYPE declaration and HTML setup:

    • <!DOCTYPE html>: Declares the document type as HTML5, which is required for modern web browsers.
    • <html lang="en">: Specifies the language of the document is English.
    • <head>: Contains metadata about the document, like character set and viewport settings for responsiveness.
    • <meta charset="UTF-8">: Sets the character encoding to UTF-8, which supports a wide range of characters.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures the game board adjusts to fit on different screen sizes (for mobile responsiveness).
    • <title>Snake Game</title>: Sets the title of the webpage.
    • <style>: This is where all CSS (styling) is defined.
    • <script>: This contains JavaScript code that handles the game's logic and interactions.

  2. Body of the HTML:

    • <div id="game-container">: A container that holds the game board and the score display.
    • <div id="game-board"></div>: The area where the snake moves and the food is placed.
    • <div class="score">Score: <span id="score">0</span></div>: Displays the score. It starts at 0 and is updated when the snake eats food.

CSS (Styling)

The CSS styles the page to make it look like a simple snake game.

CSS Reset
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
  • This applies a global reset: removes default margins and padding from all elements and sets the box-sizing property to border-box, which includes padding and border in the element's total width and height.
Body Styles
body {
    background-color: #212121;
    font-family: Arial, sans-serif;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
    color: #fff;
}
  • body: Sets the background color of the page to dark gray, changes the font to Arial, and uses Flexbox to center the game in the middle of the screen both horizontally and vertically (justify-content: center; align-items: center).
  • height: 100vh; makes sure the body takes up the full viewport height.
  • color: #fff; changes text color to white.
Game Container
#game-container {
    display: flex;
    justify-content: center;
    align-items: center;
    position: relative;
}
  • This centers the game board (#game-container) within the screen.
Game Board
#game-board {
    width: 300px;
    height: 300px;
    border: 2px solid #fff;
    display: grid;
    grid-template-columns: repeat(10, 1fr);
    grid-template-rows: repeat(10, 1fr);
    gap: 1px;
}
  • #game-board: This is where the game is drawn. It’s 300px by 300px in size.
  • display: grid: Uses CSS Grid to divide the area into 10x10 cells (for a total of 100 cells).
  • grid-template-columns and grid-template-rows: Create 10 equal-width columns and rows (each 1fr, which means one fraction of the space).
  • gap: 1px; ensures a small space between the cells.
Cell Styles
.cell {
    width: 30px;
    height: 30px;
    background-color: #2e2e2e;
    transition: background-color 0.1s ;
}
  • .cell: Styles each individual grid cell.
  • background-color: #2e2e2e;: Sets a dark gray color for the cells.
  • transition: background-color 0.1s;: Adds a smooth transition when the color of a cell changes (e.g., when the snake moves or food is placed).
Snake Styles
.snake {
    background-color: #00ff00;
}
  • .snake: This class will be applied to the cells occupied by the snake, coloring them green (#00ff00).
Food Styles
.food {
    background-color: #ff0000;
}
  • .food: This class is used to color the food red (#ff0000).
Game Over Styles
.game-over {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    font-size: 24px;
    color: #fff;
    text-align: center;
}
  • .game-over: Displays a "Game Over" message when the game ends, centered on the screen.
Score Styles
.score {
    font-size: 18px;
    position: absolute;
    top: 10px;
    left: 10px;
    z-index: 10;
}
  • .score: Styles the score text to appear at the top-left corner of the screen, in white and with a font size of 18px.

JavaScript (Game Logic)

Now we dive into the JavaScript code that controls the game’s behavior.

Initial Variables
const board = document.getElementById('game-board');
const scoreElement = document.getElementById('score');
const gameOverMessage = document.createElement('div');
gameOverMessage.classList.add('game-over');
gameOverMessage.innerHTML = 'Game Over! Press R to Restart';
  • board: Gets the game board element (#game-board).
  • scoreElement: Gets the score element (#score).
  • gameOverMessage: Creates a new div that will show the "Game Over" message when the game ends.
Game Variables
let snake, direction, food, gameInterval, score;
let nextDirection = { x: 0, y: 0 }; // Keep track of the next direction
  • Initializes variables:
    • snake: Stores the snake’s position as an array of coordinates.
    • direction: The current movement direction of the snake.
    • food: Stores the food’s position.
    • gameInterval: Used for the game loop.
    • score: Keeps track of the player’s score.
    • nextDirection: Stores the next direction the snake will move in.
    • Create Board Function
    function createBoard() {
    board.innerHTML = '';
    for (let i = 0; i < 100; i++) {
        const cell = document.createElement('div');
        cell.classList.add('cell');
        board.appendChild(cell);
    }
}
    • createBoard(): Clears the board and creates 100 cells (10x10 grid). Each cell is a div with the class cell.
    Reset Game Function
    function resetGame() {
    snake = [{ x: 2, y: 2 }];
    direction = { x: 0, y: 0 };
    food = { x: 5, y: 5 };
    score = 0;
    scoreElement.textContent = score;
    createBoard();
    generateFood();
    drawSnake();
    drawFood();
    clearInterval(gameInterval);
    gameInterval = setInterval(updateGame, 200);
}
    • resetGame(): Initializes the game:
      • Sets the snake’s initial position (x: 2, y: 2).
      • The snake starts with no movement (direction = { x: 0, y: 0 }).
      • Places food at x: 5, y: 5.
      • Resets the score to 0.
      • Creates the game board and generates the first food.
      • Starts the game loop (every 200ms).
      • Draw Snake Function
      function drawSnake() {
    const cells = document.querySelectorAll('.cell');
    cells.forEach(cell => cell.classList.remove('snake'));
    snake.forEach(segment => {
        const index = segment.y * 10 + segment.x;
        cells[index].classList.add('snake');
    });
}
      • drawSnake(): Draws the snake by adding the snake class to the cells where the snake is located.
      Draw Food Function
      function drawFood() {
    const cells = document.querySelectorAll ('.cell');
    cells.forEach(cell => cell.classList.remove('food'));
    const foodIndex = food.y * 10 + food.x;
    cells[foodIndex].classList.add('food');
}
      • drawFood(): Places the food on the grid by adding the food class to the corresponding cell.
      Move Snake Function
      function moveSnake() {
    const head = { x: snake[0].x + nextDirection.x, y: snake[0].y + nextDirection.y };
    snake.unshift(head);
    if (head.x === food.x && head.y === food.y) {
        score += 10;
        scoreElement.textContent = score;
        generateFood();
    } else {
        snake.pop();
    }
}
      • moveSnake(): Moves the snake in the direction of nextDirection:
        • Checks if the snake eats food; if so, the score is increased, and new food is generated.
        • Removes the last segment of the snake unless food is eaten.
      Generate Food Function
      function generateFood() {
    let isFoodOnSnake = true;
    while (isFoodOnSnake) {
        food.x = Math.floor(Math.random() * 10);
        food.y = Math.floor(Math.random() * 10);
        isFoodOnSnake = snake.some(segment => segment.x === food.x && segment.y === food.y);
    }
}
      • generateFood(): Generates food at a random position, ensuring it doesn't overlap with the snake's body.
      Check Game Over Function
      function checkGameOver() {
    const head = snake[0];
    if (head.x < 0 || head.x >= 10 || head.y < 0 || head.y >= 10) {
        return true;
    }
    for (let i = 1; i < snake.length; i++) {
        if (head.x === snake[i].x && head.y === snake[i].y) {
            return true;
        }
    }
    return false;
}
      • checkGameOver(): Checks if the snake has collided with the wall or itself.
      Update Game Function
      function updateGame() {
    if (direction.x !== -nextDirection.x || direction.y !== -nextDirection.y) {
        direction = { ...nextDirection };
    }
    moveSnake();
    if (checkGameOver()) {
        clearInterval(gameInterval);
        board.appendChild(gameOverMessage);
        return;
    }
    drawSnake();
    drawFood();
}
      • updateGame(): The game loop that is called every 200ms:
        • Moves the snake and checks if the game is over.
        • If the game is over, it stops the game interval and displays the "Game Over" message.
      Handle Keydown Function
      function handleKeydown(event) {
    event.preventDefault();
    if (event.key === 'ArrowUp' && direction.y === 0) {
        nextDirection = { x: 0, y: -1 };
    } else if (event.key === 'ArrowDown' && direction.y === 0) {
        nextDirection = { x: 0, y: 1 };
    } else if (event.key === 'ArrowLeft' && direction.x === 0) {
        nextDirection = { x: -1, y: 0 };
    } else if (event.key === 'ArrowRight' && direction.x === 0) {
        nextDirection = { x: 1, y: 0 };
    } else if (event.key === 'r' || event.key === 'R') {
        resetGame();
    }
}
      • handleKeydown(): Detects arrow key presses to change the snake's direction and handles the 'r' key to reset the game.
      Event Listener
      document.addEventListener('keydown', handleKeydown);
resetGame();
      • document.addEventListener('keydown', handleKeydown): Listens for keyboard events to control the snake.
      • resetGame(): Starts the game for the first time.
