Creating Life - A Yellow Dude

Level 4: Professional Level

Pac-Man Game Code Explanation

This is an implementation of the classic Pac-Man game using HTML, CSS, and JavaScript. Below is a detailed explanation of how the code works.

HTML Structure

HTML Structure
<!DOCTYPE html>
                    <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Pac-Man Game</title>
                            <style>
                                /* CSS code here */
                            </style>
                        </head>
                        <body>
                            <div id="score">Score: 0</div>
                            <div id="winMessage">You Won! 🎉</div>
                            <canvas id="gameCanvas" width="400" height="400"></canvas>
                            <script>
                                /* JavaScript code here */
                            </script>
                        </body>
                    </html>

  1. DOCTYPE declaration and HTML setup:

    • <!DOCTYPE html>: Declares the document type as HTML5, which is required for modern web browsers.
    • <html lang="en">: Specifies the language of the document is English.
    • <head>: Contains metadata about the document, like character set and viewport settings for responsiveness.
    • <meta charset="UTF-8">: Sets the character encoding to UTF-8, which supports a wide range of characters.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures the game canvas adjusts to fit on different screen sizes (for mobile responsiveness).
    • <title>Pac-Man Game</title>: Sets the title of the webpage.
    • <style>: This is where all CSS (styling) is defined.
    • <script>: This contains JavaScript code that handles the game's logic and interactions.

  2. Body of the HTML:

    • <div id="score">Score: 0</div>: Displays the current score, starting at 0.
    • <div id="winMessage">You Won! 🎉</div>: A message that appears when the player wins the game.
    • <canvas id="gameCanvas" width="400" height="400"></canvas>: The area where the game is drawn, with a width and height of 400 pixels.

JavaScript (Game Logic)

Now we dive into the JavaScript code that controls the game’s behavior.

Initial Variables

                    const gridSize = 20;

  • canvas: Gets the game canvas element.
  • ctx: Gets the 2D rendering context for the canvas, allowing for drawing shapes and images.
  • scoreElement: Gets the score display element.
  • winMessage: Gets the win message element.
  • gridSize: Defines the size of each grid cell (20 pixels).
  • rows: Calculates the number of rows based on the canvas height and grid size.
  • cols: Calculates the number of columns based on the canvas width and grid size.
  • score: Initializes the score to 0.

Draw Pac-Man Function
function drawPacman() {
                        ctx.fillStyle = 'yellow';
                        ctx.beginPath();
                        const startAngle = pacman.direction === 'right' ? 0.25 * Math.PI
                                        : pacman.direction === 'left' ? 1.25 * Math.PI
                                        : pacman.direction === 'up' ? 1.75 * Math.PI
                                        : 0.75 * Math.PI;
                        const endAngle = pacman.direction === 'right' ? 1.75 * Math.PI
                                      : pacman.direction === 'left' ? 0.75 * Math.PI
                                      : pacman.direction === 'up' ? 1.25 * Math.PI
                                      : 0.25 * Math.PI;
                        ctx.arc(
                            pacman.x * gridSize + gridSize / 2,
                            pacman.y * gridSize + gridSize / 2,
                            gridSize / 2,
                            startAngle,
                            endAngle
                        );
                        ctx.lineTo(pacman.x * gridSize + gridSize / 2, pacman.y * gridSize + gridSize / 2);
                        ctx.fill();
                    }
  • drawPacman(): Draws Pac-Man on the canvas using the arc method to create a pie shape based on its direction.
Draw Dots Function
function drawDots() {
                        ctx.fillStyle = 'white';
                        dots.forEach(dot => {
                            ctx.beginPath();
                            ctx.arc(
                                dot.x * gridSize + gridSize / 2,
                                dot.y * gridSize + gridSize / 2,
                                gridSize / 8,
                                0,
                                2 * Math.PI
                            );
                            ctx.fill();
                        });
                    }
  • drawDots(): Draws the dots on the canvas by iterating through the dots array and using the arc method to create small circles.

Move Pac-Man Function
function movePacman() {
                        if (pacman.direction === 'right') pacman.x++;
                        else if (pacman.direction === 'left') pacman.x--;
                        else if (pacman.direction === 'up') pacman.y--;
                        else if (pacman.direction === 'down') pacman.y++;
                    
                        if (pacman.x < 0) pacman.x = cols - 1;
                        if (pacman.x >= cols) pacman.x = 0;
                        if (pacman.y < 0) pacman.y = rows - 1;
                        if (pacman.y >= rows) pacman.y = 0;
                    }
  • movePacman(): Updates Pac-Man's position based on its current direction, allowing it to wrap around the canvas edges.
Move Ghosts Function
function moveGhosts() {
                        ghosts.forEach(ghost => {
                            if (ghost.direction === 'left') ghost.x--;
                            else if (ghost.direction === 'right') ghost.x++;
                            else if (ghost.direction === 'up') ghost.y--;
                            else if (ghost.direction === 'down') ghost.y++;
                    
                            if (ghost.x < 0) ghost.x = cols - 1;
                            if (ghost.x >= cols) ghost.x = 0;
                            if (ghost.y < 0) ghost.y = rows - 1;
                            if (ghost.y >= rows) ghost.y = 0;
                    
                            // Change direction randomly
                            if (Math.random() < 0.1) {
                                const directions = ['left', 'right', 'up', 'down'];
                                ghost.direction = directions[Math.floor(Math.random() * directions.length)];
                            }
                        });
                    }
  • moveGhosts(): Updates each ghost's position based on its direction, allowing them to wrap around the canvas edges and change direction randomly.
Check Dot Collision Function
function checkDotCollision() {
                        for (let i = 0; i < dots.length; i++) {
                            if (dots[i].x === pacman.x && dots[i].y === pacman.y) {
                                dots.splice(i, 1);
                                score++;
                                scoreElement.textContent = `Score: ${score}`;
                                break;
                            }
                        }
                    
                        if (dots.length === 0) {
                            winMessage.style.display = 'block';
                            cancelAnimationFrame(animationFrameId);
                        }
                    }
  • checkDotCollision(): Checks if Pac-Man has collided with any dots, removes the dot if collected, updates the score, and displays a win message if all dots are collected.

function checkGhostCollision() {

Game Loop Function
let animationFrameId;
                    function gameLoop() {
                        ctx.clearRect(0, 0, canvas.width, canvas.height);
                    
                        drawDots();
                        drawGhosts();
                        movePacman();
                        moveGhosts();
                        drawPacman();
                        checkDotCollision();
                        checkGhostCollision();
                    
                        animationFrameId = setTimeout(() => {
                            requestAnimationFrame(gameLoop);
                        }, 150); // Slower speed
                    }
  • gameLoop(): The main game loop that clears the canvas, draws the dots and ghosts, moves Pac-Man and the ghosts, checks for collisions, and requests the next frame.
Keydown Event Listener
window.addEventListener('keydown', (e) => {
                        if (e.key === 'ArrowRight' && pacman.direction !== 'left') pacman.direction = 'right';
                        if (e.key === 'ArrowLeft' && pacman.direction !== 'right') pacman.direction = 'left';
                        if (e.key === 'ArrowUp' && pacman.direction !== 'down') pacman.direction = 'up';
                        if (e.key === 'ArrowDown' && pacman.direction !== 'up') pacman.direction = 'down';
                    });
  • keydown event listener: Listens for arrow key presses to change Pac-Man's direction, ensuring it cannot reverse direction directly.
Start Game Loop
gameLoop();
  • gameLoop(): Starts the game loop for the first time, initiating the gameplay.
