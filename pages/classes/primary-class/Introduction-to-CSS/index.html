Introduction to CSS and Styling

Level 0 : The Foundation Level

Introduction

Now, just picture a site striking so entirely mundane that it's devoid of colors, layouts, and styles—anything resembling a site. That's where CSS Level 0 comes! Like the palette of an artist, CSS turns plain text and images into a user-friendly website.

The Need For CSS

CSS is the styling engine behind almost every site you visit. Here are the reasons it becomes inescapable:

  • Separation of Style and Content: With CSS, your HTML can stay out of the way, focusing on structure, while CSS handles all the visuals.
  • Design Personalization: CSS allows for the development of unique styles consistent throughout the entire website.
  • Websites Are Now Responsive: CSS design changes can now easily fit any device with screens of different widths, thus ensuring your website still looks appealing.

Groovy History of CSS

CSS was born in 1994 when Håkon Wium Lie happened to be working in CERN (the birthplace of World Wide Web) and wanted a simple way of styling web pages. The first official version of CSS was released in 1997, and since then it has seen massive adoption, with CSS3 introducing features such as animations and responsive design.

CSS—How It Works

Fundamentally, CSS contains rules that apply to HTML elements. Each rule has two components:

  • A selector: Identifies the element to style.
  • A declaration block: Defines how to style it.

For example, this rule makes all <h1> headings blue and centers them:

h1 { color: blue; text-align: center; }

Why CSS?

There are indeed unique features in CSS, which makes this tool very powerful:

  • Cascading: The styles are applied in a certain order and allow flexibility and design control.
  • Versatile: From simple text styling to very complex layouts and animations, CSS can do them all.
  • Ease of use: A few lines of code can cause huge changes to your webpage.

A Peek Into Styling

CSS can do the following:

  • Colors and Fonts: Define visual identities with distinctive colors and text styles.
