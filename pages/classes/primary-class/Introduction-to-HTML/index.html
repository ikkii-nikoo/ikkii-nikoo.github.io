Introduction to HTML

HTML (HyperText Markup Language) is the standard language for writing web pages. It organizes the contents of a web page and tells the browser which elements to display and how to organize them. HTML is not a programming language; it is a markup language that defines the structure of your content.

  • HTML Tags: HTML uses tags to define each kind of element, such as headings, paragraphs, links, and images. Tags are enclosed in angle brackets, for example, <h1> for a top-level heading, and </h1> to close the tag.

When and Who Invented HTML?

Tim Berners-Lee developed HTML for publication and use on the internet in 1991. He is also credited with the invention of the World Wide Web. With HTML, a source in the form of content is structured, thus eliminating the need for a clash between a user and a web browser such that respective demands can be met from both sides, whereas the browser understands how to process that embedded information as formatted text, images, links, and other forms of media, taking care of other multimedia.

Learn more at: HTML Coding History

HTML Structure and Layout

Having a handle on the composition of an HTML document is crucial in building functional web pages. An HTML document strictly follows a layout setting, which has a specific DOCTYPE declaration, an HTML tag, a head, and a body.

  • DOCTYPE Declaration: The <!DOCTYPE html> informs the browser what version of HTML is in use. With this, the browser can properly render web pages.
  • HTML Tag: This <html> is the root element of an HTML document. It comprises any content present on the webpage.
  • Head Section: The <head> will store that metadata-informative details from the page that never presents directly within the page. These things include page title, character encoding, links to stylesheets or scripts, and other items.
  • Body Section: The <body> element is the actual content of the page, and it usually consists of text and pictures, with links to other pages. Anything found within the <body> tag is displayed in the browser.

Working for Practice

The working practice for HTML practice is as follows:

  1. Construct a simple HTML document: It must be meaningfully structured.
  2. Attach a picture along with a hyperlink to your favorite website.
  3. Keep the file with a .html extension as filename and open it in a browser to see your work!
