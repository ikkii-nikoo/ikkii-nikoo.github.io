Introduction to Web Development

This interactive learning experience teaches HTML and CSS fundamentals for creating websites, enhancing skills for beginners or those looking to enhance their skills.

This module teaches website structure, style, and basic web page creation through interactive exercises, quizzes, and real-world examples, preparing students for web development through HTML and CSS.

Lesson Overview

This lesson is divided into three main sections that will help you understand the basics of web development:

  • HTML Basics: Learn the fundamental structure of HTML and how it is used to create the content of a webpage. We will explore the basic tags, such as headings, paragraphs, links, images, and lists, and discuss how HTML provides structure to a webpage.
  • CSS Styling: Once you understand HTML, you will learn how to style your webpage using CSS. You'll discover how to change the color, layout, and fonts of your webpage to make it visually appealing.
  • Building a Simple Webpage: Putting it all together, you'll apply what you've learned to create your very first webpage. This will include writing HTML to structure the content and using CSS to enhance the design.

HTML

This section will dive into HTML, explaining its role in web development and the essential tags you need to know.

Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures the content of a webpage, telling the browser what elements to display and how to organize them. HTML is not a programming language, but a markup language that defines the structure of your content.

  • HTML Tags: HTML uses tags to define elements such as headings, paragraphs, links, and images. Tags are enclosed in angle brackets, for example, <h1> for a top-level heading, and </h1> to close the tag.

Who Created HTML?

HTML (HyperText Markup Language) was created by Tim Berners-Lee, a British computer scientist, in 1991. He is also the inventor of the World Wide Web. HTML was developed as a way to structure and display content on the internet, making it possible for browsers to understand and present text, images, links, and other media in a formatted way. Berners-Lee’s work in developing HTML and the web is considered one of the most significant contributions to the digital age, shaping how we access and share information online.
